HDW Tube is an impressive plugin for WordPress websites. This plugin is altogether different and unique from other plugins available in Wordpress. You can effortlessly install this plugin to your WordPress website on the go. Most importantly through this plugin, you can create webpages as exact as youtube (as you see in youtube.com) with features which come with package. You would love this plugin for the look and the versatile options that comes with the package. Start downloading the plugin either from WordPress or from our website and save or the never-ending video experience.

Features include:

* HDW Tube can handle variety of Media formats that flash can handle such as flv, mp4, 3g2, 3gp, aac, f4b, f4p, f4v, m4a, m4v, mov(h.264), sdp, vp6.	
* The Plugin includes native support for playing back Youtube videos. It can be done by just providing Youtube page URL to the plugin.
* The Plugin includes native support for playing back Dailymotion videos. It can be done by just providing Dailymotion page URL to the plugin.
* RTMP stands for "Real Time Messaging Protocol". It is a proprietary protocol developed by Adobe Systems for streaming audio, video and data over the Internet, between a Flash Player and a server. All the major RTMP servers such as Red5, Wowza and Adobe Flash Media Server are supported by HDW Tube.
* If you are using a commercial version with the KEY provided by HDW Tube Sales Team, you will have an option to remove HDW Tube FREE text and you will be having your own Domain Name in the Context Menu that comes from the Right Click on the Player.
* With our HDW Tube, you have freedom of enabling / disabling all the features that you can read in detail from our Documentation part.
* Using HDW Tube, you can embed your videos on other websites easily by using the code generated on clicking Share - Embed Button.
* HDW Tube has built-in option to share the video to the famous Social Networking sites such as Facebook, Twitter, etc...
* HDW Tube allows you to share the video (that impressed you) to your friend by using the Email option inside the Player.
* Button Tooltip is available for each button on the Player.
* As youtube does, HDW Tube does also have the facility of adding channels, adding videos to channel.

== Installation ==

Installing HDW Tube plugin can be achieved in two easy methods:

1. Uploading Manually via WordPress Admin.
2. Uploading Manually via FTP.

Uploading Manually via WordPress Admin :

1. Download the latest package.
2. From the WordPress plugin menu click on Add New.
3. Under the Upload menu, Use the Browse button to select the plugin zip file that was downloaded, then click on Install Now. The plugin will be uploaded to your site and installed. It can then be activated.

Uploading Manually via FTP :

1. Download the latest package and unzip the plugin.
2. Now you will need to upload the plugin to your site's wp-content/plugins/ directory using FTP.
3. Congrats! You have installed the plugin. It can then be activated.

== Usage ==

Adding HDW Tube to your WordPress require three necessary steps :

#1 : Configuration

1. Navigate to Configuration menu.
2. Provide page Settings under Front-end Tab.
3. Configure the basic player options under Player Tab
4. Configure the Upload and Video settings under Upload Tab
5. Configure the server details under Server Tab

#2 : Category

1. Navigate to Category Menu.
2. Click on the Add New button.
3. Give category name and click save button.
4. Congrats! Now you have created the category.

#3 : Channels

1. Navigate to Channels menu.
2. Click on the Add New Channel button.
3. Fill Channel Settings and save.

#4 : Playlists

1. Navigate to Playlists menu.
2. Click on the Add New button.
3. Fill Playlist Settings and save.

#5 : Videos

1. Navigate to Videos menu.
2. Click on the New Video button.
3. Fill Video Settings and save.

#6 : Recommended

1. Navigate to Recommended menu.
2. Provide Channel ID for recommended channels.
3. These Channels are shows in front page.

#8: HDW Tube Widget

1. Navigate to Widgets menu.
2. Drag and Drop the HDW Tube Widget on the widget area.
3. Provide HDW Tube URL and Logo then save.

Note : Add plugin shortcode [hdwtube] to wordpress page or post
